Accessibility for the modern web developer

It is both possible and practical to build modern websites and applications that are accessible to all potential users, including those with disabilities. We aim to teach you how.

Accessibility Shortcuts

  • Form elements require labels.

  • Images require alternative text.

  • Prefer button to div.

Further Reading

  • The W3C's Web Content Accessibility Guidelines are solid recommendations to developers and authors. Many of w15y.com's principles are rooted in this document. It's long and involved, however: the WCAG Overview might be useful.
  • Techniques for Providing Useful Text Alternatives is a great, and incredibly detailed discussion of when and how to use alternative text in an HTML document. Very practical work indeed, and worth using as a reference.
